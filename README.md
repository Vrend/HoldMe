# HoldMe
Decentralized Storage - Inspired by Hivemind


## Dependencies

1. flask
2. redis-py
3. passlib

## How to Use

Clone the repository and install redis

Run redis with the command `redis-server HoldMe/redis.conf`

Then run `python3 app.py`
